# AI-Powered Marketing Recommendation System

This repository contains my Master's thesis project developed during my MSc in Data Science at the University of Europe for Applied Sciences.

## Project Overview

The goal of this project is to help businesses identify the most effective social media platforms for advertising different product categories using machine learning techniques.

The recommendation system evaluates marketing performance indicators and generates data-driven recommendations to improve advertising efficiency.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Methodology

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Correlation Analysis
- Feature Importance Analysis
- Random Forest Machine Learning Model
- Marketing Recommendation Generation

## Dataset

Social Media Advertising Dataset (Kaggle)

Features include:

- ROI
- Conversion Rate
- Clicks
- Impressions
- Engagement Score
- Acquisition Cost
- Customer Segments

## Key Findings

- Engagement Score was one of the strongest predictors of campaign success.
- AI-driven recommendations can support better allocation of marketing budgets.
- Businesses can improve targeting efficiency and reduce advertising waste through data-driven decision making.
